Backend

NOTE:-I already deployed a backend and database on aws using by my personal credential.
you dont have to run the backend sepratley.
But if you want to run the backend locally you have to install the xampp locally to run the database server.

Dont forget to change the IP of Your machine in frontend 
assests>base_url.js

If you want to run the application locally.

Otherwise it will connected with aws public IP on PORT 8080.

Steps to Run the backend.

1.Clone the backend in your machine using github.
2.Go to the folder and hit the command npm i.
3.After succesfully run the command hit the command "npm start" or "nodemon app.js" or "node app.js".



NOTE:dont forget to run the xampp server  if you want to test database with your app.
